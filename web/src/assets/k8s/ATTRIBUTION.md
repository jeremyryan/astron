# Kubernetes resource icons

The SVG files in this directory are the **official Kubernetes resource
icons**, vendored from the [`kubernetes/community`](https://github.com/kubernetes/community/tree/master/icons)
repository (`icons/svg`, the *unlabeled* variants).

- **Source:** https://github.com/kubernetes/community/tree/master/icons
- **Copyright:** The Kubernetes Authors
- **License:** Apache License, Version 2.0

They are used in the Project Gamera web UI to visually identify the
corresponding Kubernetes resource kinds (Pod, Deployment, Service,
ConfigMap, Secret, etc.).
